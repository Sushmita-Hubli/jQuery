Events

All the different visitors' actions that a web page can respond to are called events
event represents the precise moment when something happens.eg. moving a mouse,selecting a radio button,etc...

Commonly used jQuery event methods : $(document).ready() method allows us to execute a function when the document is fully loaded

Mouse Events



contextmenu() -->function is executed when the right mouse button is clicked





mouseenter() and mouseleave()



This event is triggered when a mouse button is pressed down over an element. It fires before the mouseup event. It is often used to initiate actions like dragging or selecting elements.



This event is triggered when a mouse button is released over an element. It fires after the mousedown event. It is often used to complete actions like dragging or selecting elements.

Key differences: Timing: mousedown occurs when the button is pressed, while mouseup occurs when the button is released. Action: mousedown often initiates an action, while mouseup typically completes it. Use case: mousedown is useful for tasks like dragging, while mouseup is used to finalize those actions.



focus() event --> executed when form field gets focus



on() method attaches one or more event handlers for the selected elements
